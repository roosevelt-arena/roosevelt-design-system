7.4 — UI-microcopy voorbeelden

Toepassing van grammar.md + vocabulary.md in Roosevelt-context

Wel — canon-correct

  • ✓ UI button: «Start intake»
  • ✓ Helper text: «Je voortgang is bewaard.»
  • ✓ Lege staat: «Nog geen diagnose. Start je intake.»
  • ✓ F4-status: «F4 Strategic-kern — 67% verwerkt»
  • ✓ Foutmelding: «De pipeline kon F3 niet voltooien. Probeer opnieuw.»
  • ✓ F9-rapport: «Uw salesorganisatie scoort 34 op Dimensie 3 Proces-automatisering — het laagste van vier dimensies.»
  • ✓ Bevestiging: «Intake verwijderen?»

Niet — canon-overtreding

  • × «Start Uw Intake!» (titlecase + uitroepteken)
  • × «Je bent er bijna!» (consumer-gamification)
  • × «Oeps! Er ging iets mis» (neerbuigend + Er-opening)
  • × «Step 4 — analyzing your data» (Engels + step ipv F-prefix)
  • × «Onze AI-powered solution…» (hype + our solution)
  • × «Your enterprise heeft 3 action items» (Concentrix-taal + cijfer onder 13)
  • × «Gefeliciteerd! Uw rapport is klaar!» (uitroeptekens + overdreven)

8.5 — Layout-shifts per breakpoint

Compact · < 640px
1 koloms. Topbar collapsed naar hamburger. Sidebar → drawer. Right-rail verborgen. Body-padding 16px. Touch 44px+.
Regular-sm · 640–1023
1 koloms content, tabellen breder. Sidebar nog drawer. Right-rail nog verborgen. Body-padding 24px.
Regular-md · 1024–1279
2-koloms: sidebar 240px + main. Right-rail nog verborgen. Topbar volledig zichtbaar.
Regular-lg · 1280+
3-koloms: sidebar 240px + main + right-rail 220px. Scrollspy actief. Volledige Wise.design-stijl.
